# OrtusCapture — гайд по использованию (RU)

## Для игроков

- `/oc help` — список команд.
- `/oc info` — информация о земле Lands в текущем чанке и о регистрации OrtusCapture.
- `/oc status` — статус плагина (хуки, записи, активные захваты, день недели).
- `/oc gui` — открыть меню текущего зарегистрированного региона (если вы на его земле).

### Как начать захват

1. Получите флаг захвата (через крафт или от администратора).
2. Придите в зарегистрированный регион нужного типа.
3. Поставьте флаг в точке флагштока (если она задана админом).
4. Не покидайте территорию до завершения захвата.

### Важно про флаг

- Флаг — это предмет-баннер, но при установке появляется display-модель.
- У флага есть HP. Его можно ломать ударами.
- Ремонт: `Shift + ПКМ` рядом с флагштоком (медленный ремонт).

## Для администраторов

- `/oc setup <id> <type>` — привязать текущую землю Lands к системному ID.
- `/oc create <id> <type> [name...]` — создать Land и зарегистрировать регион.
- `/oc capture <id> <on|off>` — включить/выключить захват региона.
- `/oc point set flagpole <id>` — задать точку флагштока по вашей позиции.
- `/oc point clear flagpole <id>` — очистить точку флагштока.
- `/oc giveflag <player> <id>` — выдать флаг для региона.
- `/oc reset <id>` — сброс владельца/кулдауна региона.
- `/oc reload` — перезагрузка конфигов (`config`, `regions`, `lang`, `crafts`, `flags`) и документации.
- `/oc gui admin` — открыть админ GUI.
- `/oc gui <id>` — открыть GUI конкретного региона.

## Debug-команды (админ)

- `/oc debug admin` — временный bypass ограничений по дням (до выхода с сервера).
- `/oc debug status` — показать статус debug.
- `/oc debug off` — выключить bypass.

## Примечания

- Если `Lands` временно недоступен — часть функций будет отключена.
- Все команды и сообщения плагина настраиваются через `lang.yml`.
